# Spotify Web Player Clone
A fully functional Spotify-inspired web music player homepage built using HTML, CSS, and JavaScript.

## Features
* Sidebar navigation (Home, Liked Songs, Library)
* Centered search bar UI
* Playlist cards with cover images
* Dynamic song loading using JSON
* Play / Pause / Next / Previous controls
* Seekbar with live progress
* Volume control with custom styled slider
* Responsive layout for mobile screens
* Clean dark Spotify-style UI

## Tech Stack
* HTML5
* CSS
* JavaScript

## Project Structure
* index.html – layout
* style.css – main styling
* utility.css – helper classes
* script.js – player logic
* songs/ – playlists and audio files
